## Installation

Install packages    
`npm install`    

Update packages    
`npm update`    

## Scripts

1. `build`    
Transpile ES6 to **build** directory.    

2. `dist`    
Transpile and minify ES6 to **dist** directory.    

3. `test`    
Run tests in `test` firectory.    

4. `start`
Run the app
